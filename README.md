## Installation

- Download the latest installable ZIP file from: https://estudiopatagon.com/projects/slate-for-ghost/
- Login into your Ghost Admin and upload into Themes section and "activate it"
- Customize the Colors, sections and Typography on Ghost admin -> Design & Branding
