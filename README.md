## Installation

- Download the latest installable ZIP file from our: [Official Project Page](https://estudiopatagon.com/projects/slate-lite-for-ghost//?utm_source=readme)
- Login into your Ghost Admin and upload into Themes section and "activate it"
- Customize the Colors, sections and Typography on Ghost admin -> Design & Branding

## Features

- **Ghost 6.x Compatible**
- **Highly optimized for speed.**
- **Accent & Background is customizable from admin.**
- **Custom Author Section on Homepage.**
- **Custom Featured Post on Homepage.**
- **Modern & Clean design.**
- **Native Search.**
- **Native Comments.**
- **Secondary Navigation support (on Footer).**
- **Full Koenig Editor Support, check [demo here.](https://ghost.estudiopatagon.com/slate/style-guide/).**

- **Constantly updated.**
- **Translation ready (English and Spanish Included).**
- **Minified and Optimized code.**
- HTML5 & CSS3 ready.
- Custom Tag page, [see here](https://ghost.estudiopatagon.com/slate/tag/inspiration/).

- Works on Android, Ipad and Iphone.
- Well Coded.
